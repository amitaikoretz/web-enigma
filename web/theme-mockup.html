Reference mashup

Apple HIG + Fluent 2 theme candidates

Ten mockups that mix Apple’s hierarchy, rounded interaction affordances, and translucent materials with Fluent 2’s spacing, elevation, and clean surface language. The revised set pushes each palette into a distinct direction so the options feel more original, expressive, and memorable.
